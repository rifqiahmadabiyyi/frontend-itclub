Divisi Programming

Divisi Programming adalah suatu satu divisi yang ada IT CLUB yeng mempelajari tentang ilmu pemograman. Nah, programming itu sendiri berasal dari kata program. Program adalah sebuah urutan logika yang digunakan untuk menghasilkan keluaran tertentu yand disatukan menjadi sebuah sistem utuh yang disebut aplikasi. Dalam Divisi programming, kita akan berkutak pada script-script atau source code maupun bahasi pemograman. Keuntungan bergabung di Divisi Promgramming ialah kita bisa membuat suatu website sendiri, juga mengetahui bahasa pemograman agar bisa menjadi seorang programer yand handal.
